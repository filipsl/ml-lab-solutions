This is the repository for third jupyter notebook tutorial from Machine Learning classes
at AGH University of Science and Technology.

---
## Launching notebook
### 1. Set up environment:
#### Windows:
```
setup.bat
```

#### Linux & MacOS:
```
 ./setup.sh
```

### 2. Launch Jupyter Notebook:
#### Windows:
```
run.bat
```

#### Linux & MacOS:
```
 ./run.sh
```
